- **0.1.1.2 Конвертация диска VMware → Hyper-V (VHDX)**
  - **0.1.1.2.1** Выбрать способ конвертации:
    - **0.1.1.2.1.1** Через конвертер на Windows (например, StarWind V2V) — вход: `.vmdk`, выход: `.vhdx`.
    - **0.1.1.2.1.2** Либо через `qemu-img` (если удобнее) — вход: `.vmdk`, выход: `.vhdx`.
  - **0.1.1.2.2** ✅ Сконвертировать основной диск(и) ВМ из `...\w10-26-0310\*.vmdk` в новый каталог, например: `D:\Program_Files\vm\hyper-v\w10-26-0310\disk0.vhdx` (и `disk1.vhdx`, если есть).

    - **Примечание:** выполнен бэкап исходной папки ВМ: `D:\Program_Files\vm\vmware\w10-26-0310_BACKUP` (robocopy, `FAILED=0`), затем удалены файлы suspend (`*.vmss`, `*.vmem`) из исходной папки перед конвертацией.
